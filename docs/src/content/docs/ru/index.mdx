Xray Checker - это комплексное решение для мониторинга, предназначенное для проверки и отслеживания доступности прокси-серверов, работающих по различным протоколам, включая VLESS, VMess, Trojan и Shadowsocks. Разработанный с учетом современных практик DevOps, он легко интегрируется с популярными системами мониторинга и предоставляет метрики в реальном времени для вашей прокси-инфраструктуры.

## Что такое Xray Checker?

Xray Checker служит мостом между вашими прокси-серверами и инфраструктурой мониторинга, выполняя следующие функции:

- Непрерывное тестирование прокси-соединений через Xray Core
- Предоставление метрик Prometheus для детального мониторинга
- Предоставление конечных точек статуса для мониторинга доступности
- Поддержка автоматического обновления конфигурации из URL-адресов подписок
- Представление чистого веб-интерфейса для быстрой проверки статуса

Представьте его как монитор здоровья вашей прокси-инфраструктуры - постоянно проверяющий пульс, сообщающий о жизненных показателях и предупреждающий вас, когда что-то требует внимания.
